  • 23 April - GeoPandas for geospatial data in Python

    GeoPandas is a project that started in 2013 and incorporates some of the most useful geospatial data processing packages available in Python. It’s very comprehensive but a bit on the slow side because it stores custom Python objects in arrays, so for operations it has to iterate through those objects. Some recent enhancements with Cython means it’s closing the gap to bigger open-source tools like PostGIS.
